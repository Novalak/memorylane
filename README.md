# MemoryLane

A beautiful memorial website for sharing photos with slideshow functionality.

## Quick Start

```bash
git clone https://github.com/Novalak/memorylane.git
cd memorylane
docker-compose up -d
```

Access at: http://localhost:4173

- Admin password: `admin123`
- Upload photos with drag & drop
- Full-screen slideshow with customizable effects
- Export all photos as ZIP

## Features

- 📸 Photo upload with progress tracking
- 🎬 Professional slideshow with multiple transitions
- ⚙️ Admin panel for configuration
- 📱 Responsive design
- 🐳 Docker ready
